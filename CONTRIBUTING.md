# 기여 가이드

이 프로젝트에 기여해주셔서 감사합니다!

## 시작하기

1. 이 레포를 Fork합니다
2. 새 브랜치를 만듭니다 (`git checkout -b feat/your-feature`)
3. 변경 사항을 커밋합니다 (`git commit -m "feat: 기능 설명"`)
4. 브랜치를 Push합니다 (`git push origin feat/your-feature`)
5. Pull Request를 엽니다

## 코드 스타일

- Python: PEP 8 준수
- 노트북 셀: 실행 전 Output 초기화 후 커밋
- 환경 변수: `.env`는 절대 커밋하지 않습니다 (`.env.example`만 사용)
